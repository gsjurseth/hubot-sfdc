# hubot-sfdc

A hubot script that does the things

See [`src/sfdc.coffee`](src/sfdc.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-sfdc --save`

Then add **hubot-sfdc** to your `external-scripts.json`:

```json
[
  "hubot-sfdc"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
